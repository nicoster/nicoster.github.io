大家都知道， win2000pro 下在控制面板里面有一个 applet 可也设置 windows 自动登录。
但在 server 版下， 却没有这个选项了。
其实可也用这个命令调出那个 applet.
windows2000: 
control userpasswords
windows2003:
control userpasswords2
